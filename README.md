## Installation and Setup
1. Clone the repository
```bash
git clone https://github.com/CusnirovMaxim/portfolio.git
```

2. Install dependencies
```bash
cd portfolio
npm install
```

3. Run development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```
